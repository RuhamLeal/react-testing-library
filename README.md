## Getting Started

To acess the tests, just need to clone the repository:

```
git clone https://github.com/RuhamLeal/js-hofs-es6-functions.git
```
```
cd js-hofs-es6-functions
```

If you want to see de complete aplication, install dependicies:

```
npm install
```
```
npm start
```
